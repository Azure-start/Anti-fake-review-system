## 技术栈实现详情

### 核心依赖
| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.5.22 | 前端框架 |
| Vue Router | 4.6.3 | 路由管理 |
| Pinia | 3.0.3 | 状态管理 |
| Element Plus | 2.11.5 | UI组件库 |
| Ethers.js | 6.x | 区块链交互 |
| Axios | 1.12.2 | HTTP请求 |

### 项目结构
```
src/
├── api/               # API接口封装（5个文件）
├── components/        # 公共组件（4个组件）
├── stores/           # Pinia状态管理（3个store）
├── utils/            # 工具函数
├── views/            # 页面组件（8个页面）
├── router/           # 路由配置
├── assets/           # 静态资源
├── App.vue           # 根组件
└── main.js           # 入口文件
```

## 核心页面列表

1. **Home.vue** - 首页，商品列表展示
2. **Login.vue** - 登录页，MetaMask连接
3. **ProductDetail.vue** - 商品详情，评价预览
4. **Checkout.vue** - 交易确认，支付流程
5. **Transactions.vue** - 交易记录列表
6. **Review.vue** - 评价发布页面
7. **Rewards.vue** - 奖励记录，优惠券兑换
8. **Account.vue** - 个人账户，统计信息

## 核心组件列表

1. **WalletConnectBtn** - 钱包连接按钮
2. **NavBar** - 顶部导航栏
3. **ProductCard** - 商品卡片组件
4. **NftTip** - NFT提示弹窗
5. **RewardList** - 奖励列表组件

## API接口封装

1. **request.js** - Axios配置与拦截器
2. **authApi.js** - 认证接口（nonce、签名登录）
3. **productApi.js** - 商品接口（列表、详情、评价）
4. **transactionApi.js** - 交易接口（创建、查询）
5. **rewardApi.js** - 奖励接口（余额、记录、兑换）

## 状态管理

1. **userStore** - 用户钱包信息、余额、登录态
2. **productStore** - 商品列表、当前商品
3. **chainStore** - 区块链网络状态
