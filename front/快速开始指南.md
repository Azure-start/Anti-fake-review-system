# 区块链电商防刷评系统 - 快速开始指南

## 🚀 快速开始

### 第一步：检查环境

确保已安装：
- ✅ Node.js >= 20.19.0 或 >= 22.12.0
- ✅ npm 或 yarn
- ✅ Chrome/Firefox/Brave/Opera 浏览器
- ✅ MetaMask 浏览器插件

### 第二步：安装依赖

```bash
cd D:\develop\Vue\Anti-fake-review-system
npm install
```

### 第三步：配置环境变量

创建 `.env.development` 文件（如果不存在）：

```env
# 后端API地址
VITE_API_BASE_URL=http://localhost:3000/api

# 区块链网络配置
VITE_CHAIN_ID=11155111
VITE_CHAIN_NAME=Sepolia
```

### 第四步：启动项目

```bash
npm run dev
```

访问 http://localhost:5173

## 🔧 常见问题排查

### 问题1：页面显示空白或报错

**检查清单**：
1. 确认依赖已安装：`npm list --depth=0`
2. 清除缓存重新安装：`rm -rf node_modules package-lock.json && npm install`
3. 检查浏览器控制台错误信息

### 问题2：连接钱包失败

**解决方案**：
1. 确认已安装 MetaMask 插件
2. MetaMask 已解锁
3. 切换到 Sepolia 测试网络
4. 浏览器允许弹出窗口

### 问题3：API请求404错误

**原因**：后端服务未启动

**解决方案**：
1. 启动后端服务（端口3000）
2. 或修改 `.env.development` 中的 `VITE_API_BASE_URL` 为实际后端地址
3. 检查 Vite 代理配置是否正确

### 问题4：Element Plus 图标不显示

**已经修复** ✅

如果仍遇到问题，检查 `src/main.js` 中是否包含：
```javascript
import * as ElementPlusIconsVue from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
  app.component(key, component)
}
```

### 问题5：构建失败

**解决方案**：
1. 清除 dist 目录：`rm -rf dist`
2. 重新构建：`npm run build`
3. 检查 Node.js 版本：`node -v`
